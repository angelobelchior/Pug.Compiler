**Funcionalidade:**

Nessa branch temos o motor de interpretação de expressões numéricas simples:

Podemos executar as seguintes expressões:

```bash
> 1 + 2
3

> 1 + (2 * 3)
7

> (2 * (3 * 4) / 5) + 6
10,8
```
